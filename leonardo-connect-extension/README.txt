Leonardo Connect — GenityBoost browser extension
=================================================

This extension links your Leonardo.AI account to your GenityBoost account so its
token balance joins your pool. It only reads the session your browser already
stored after you log in — no password, no keystrokes.

INSTALL (Chrome / Edge / Brave)
-------------------------------
1. Unzip this file into a folder (keep all files together).
2. Open chrome://extensions  (or edge://extensions).
3. Turn on "Developer mode" (top-right).
4. Click "Load unpacked" and select the unzipped folder.

USE
---
1. Log in at https://app.leonardo.ai
2. Click the "Leonardo Connect" extension icon.
3. The server is fixed to GenityBoost (https://api.genityboost.site) and cannot be changed.
4. Paste your Connect token (copy it from your GenityBoost dashboard).
5. Click "Connect this account". You'll see your email, plan and tokens left.
6. Refresh your dashboard — the Leonardo account now shows up.

RE-CONNECT
----------
Only needed if the ~45-day session expires, you log out, or change your password.
Otherwise the server keeps the session alive automatically.
